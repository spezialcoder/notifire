Zuerst erstellen sie einen neuen user mit "python nadduser.py <name> <role> <password>" Bei Role geben sie für ein administrator recht "admin.role" ein für ein user recht "user.role"

Nun configurieren sie notifire.conf
Ein beispiel

#Configuration of notifire 
port = <Port zum verbinden>
#denied_user = ["user1","user2"] 
denied_user = <Verbotene benutzer (für keinen bitte "None" eintragen>
server_name = <Ihr name für den server>
listen = <Wie viel leute verbinden sich>
ip = <Ip vom server>

First, create a new user with "python nadduser.py <name> <role> <password>" For Role, enter "admin.role" for an admin right for "user.role"

Now configure notifire.conf
An example

#Configuration of notifire
port = <port to connect>
#denied_user = ["user1", "user2"]
denied_user = <Forbidden users (for none please enter "None">
server_name = <your name for the server>
listen = <How many people connect>
ip = <ip from server>

Now you still have to configure the client. When creating a new user, the secret will be output at the end. You have to copy this (without "Secret:") and insert an example into the client configuration

#Configuration of notifire
port = <port to connect>
ip = <ip to connect>
secret = 4d4152436e6a4a656b5551587463504d7959766e47587854656159567444

Now you can start notifire with "python notifire.py"
